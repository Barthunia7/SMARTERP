# SmartERP - Streamlined MVP Scope Specifications

## 1. Focused Voucher Models
* **Purchase Voucher**: Logs stock incoming from suppliers. Automatically increments current warehouse item totals.
* **Sales Voucher**: Handles outgoing customer billing. Automatically decrements current warehouse item totals.

## 2. Master Module Properties
* **Ledgers**: Limited to Supplier Ledgers and Customer Ledgers.
* **Stock Items**: Tracks Name, SKU, Purchase Price, Selling Price, and current stock balance.

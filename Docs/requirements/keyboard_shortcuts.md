# SmartERP Complete Keyboard Shortcut Architecture

## 1. Global Navigation Shortcuts
* `F1`: Company Selection
* `F2`: Change Financial Year
* `F3`: Company Information
* `F4`: Calculator
* `F5`: Refresh
* `ESC`: Previous Screen
* `CTRL + Q`: Logout
* `CTRL + H`: Home
* `CTRL + K`: Command Search

## 2. Masters Shortcuts
* `ALT + L`: Create Ledger
* `ALT + A`: Alter Ledger
* `ALT + G`: Create Group
* `ALT + S`: Create Stock Item
* `ALT + U`: Unit Creation

## 3. Voucher Shortcuts
* `F6`: Receipt Voucher
* `F7`: Journal Voucher
* `F8`: Sales Voucher
* `F9`: Purchase Voucher
* `F10`: Reversing Journal
* `ALT + F8`: Credit Note
* `ALT + F9`: Debit Note

## 4. Inventory Shortcuts
* `CTRL + I`: Inventory Dashboard
* `CTRL + N`: New Item
* `CTRL + E`: Edit Item
* `CTRL + D`: Delete Item
* `CTRL + T`: Stock Transfer
* `CTRL + R`: Stock Report

## 5. Billing Shortcuts
* `CTRL + B`: New Invoice
* `CTRL + P`: Print Invoice
* `CTRL + SHIFT + P`: PDF Download
* `CTRL + M`: Email Invoice

## 6. Stakeholder Shortcuts
* `CTRL + C`: New Customer
* `CTRL + SHIFT + C`: Customer Ledger
* `CTRL + S`: New Supplier
* `CTRL + SHIFT + S`: Supplier Ledger

## 7. Reports Shortcuts
* `ALT + B`: Balance Sheet
* `ALT + P`: Profit & Loss
* `ALT + T`: Trial Balance
* `ALT + C`: Cash Flow
* `ALT + R`: Stock Summary
* `ALT + X`: GST Report

## 8. Data Grid & Form Fields
* `CTRL + F`: Search
* `CTRL + SHIFT + F`: Global Search
* `ARROW KEYS`: Line Item Grid Navigation
* `ENTER`: Select Active Element
* `TAB`: Focus Next Form Field
* `SHIFT + TAB`: Focus Previous Form Field
